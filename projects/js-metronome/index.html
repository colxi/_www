JSMETRONOME : HTML5 MUSICAL METRONOME

JSMetronome is a cross-browser HTML5 musical metronome webApp. It supports a wide range of common musical time signatures, a special accent in the first beat can be enabled, and allows tapping for tempo setting. Is build on the top of a musical click engine capable of handling time subdivisions.


The source code of this project is available in Github for public download.
