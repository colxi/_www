DUSTLAND : RETRO GAME EXPERIMENT

Dustland is an experiment about retro arcade games development. It's been developed using the fantastic SCRIPT-8 platform, which provides the necessary playground for easy sprite creation and manipulation (among so many other things). The game provides some basic physic mechanics, and the necessaru input handling to let the user interact with the map and enemies.
The SCRIPT-8 engine has been pushed out of its bound in this implementation, when using (non-nativelly supported) multi-sprite characters, complex sprites animations, and a (kinda hacky) double layered map, to achieve more expressive characters, and more immersive and captivating maps. Obviously always conditioned by the limitations of an 8-bit simulated environment.

To test the game, click in its viewport to set focus, and use the ARROW KEYS for movement, A for kick, and SPACEBAR to toggle Debug mode on/off.

Note : This game implementation is just an experiment, don't expect a complete game experience. All the game code is available in the CODE TAB, and the sprites and maps can be found in the ART TAB.
