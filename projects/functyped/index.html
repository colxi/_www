FUNCTYPED : STATIC TYPES FOR JS FUNCTIONS

Functyped is a library for automatic runtime static type-checking in javascript function parameters, through declarative syntax. It provides a complete support to the most common types.
Is highly performant, multi enviroment & crossbrowser (Node, Chrome, Firefox, Safari, Edge...), and supports complex typed structures ( Objects and Arrays ). Custom types, and their corresponding type-checks can be implemented and added to the engine.
