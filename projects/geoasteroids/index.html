GEOASTEROIDS RETRO GAME

Game experiment implemented in Scratch and then ported to HTML5. The game is a geometrical clone of the retro game Asteroids, where the user needs to move arround avoiding obstacles, shooting against them, and handle the acceleration of the spaceship.

USAGE: Click on the game viewport to set the focus and use ARROW KEYS to move and SPACE BAR to shoot.
