ADVANCED TIMER JS

Javascript implementation of a reusable advanced timer, with pause/resume/reset capabilites, conditional cycle execution, onComplete tasks scheduling, live timer delay manipulation, etc...

The timer initialization is quite intuitive thanks to its chainable Constructor, and once initiated it still allows live parameter manipulation, through its extensive API, without the need to stop or reset the timer. In each cycle callback, the engine provides a complete set of data representing the timer state, including a flag to detect and identify timer Sync lost.

The source code of this project is hosted in Github, where it's available for download, and detailed documentation is provided.
