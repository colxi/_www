UMDH : SOCIAL NETWORK

UMDH is a complete and ambitious HTML5 Social Network platform. Is based in a charity concept, where users perform a micropayent to become members, supporting this way the humanitarian actions developed by the organization behind the platform. The platform provides user profiles, avatar customization, activity walls, internal messaging system, admin roles, conversation threads, and an extense list of extra features.


Fields

- Web development
- AJAX
- HTML5
- PHP, MySql
- Payment gateways
