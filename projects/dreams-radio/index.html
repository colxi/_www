DREAMS ONLINE RADIO : LOGO + WEBAPP

Dreams Online Radio Station, is an HTML5 based online radio station. Originally developed using Adobe Flash, the Radio has been reimplemented from scratch using the HTML5 standards keeping in mind the crossbrowser compatibility, and a fluid and intuitive user experience. A redesign was performed to achieve it, and a new logo made.
