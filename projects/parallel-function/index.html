PARALLEL FUNCTION : JS PARALLELIZATION

This open source library lets you create a new thread (Web Worker) with a function, and keep it alive and ready to be called as many times as you need, asynchronously in the background, without blocking your main thread event loop.
With this simple solution you can move to secondary threads your heavy functions, and enjoy multithreading in a way that feels like calling a regular async function.

Note : A ParallelFunction, feels and behaves as a regular async function, being the only downside, an architectural limitation that limits the type of the function parameters and function return value, to be in the family of values which can be handled by the Structure Clone Algorithm.

This project, and its source code is hosted in Github, where can be downloaded and complete documentation is available.
