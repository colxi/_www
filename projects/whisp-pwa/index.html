WHISP : MODERN CHAT APP

Whisp is a modern Chat App with encrypted private groups and rich messages support, based in Node (chat Server) and HTML Progressive Web Application (chat Client)

Features :
- Private Encrypted Groups (AES encryption)
- Rich messages support : Text / Audio / Images
- Multienviroment : Browser / Desktop / Devices (PWA)
- Detachable Server
- Easilly customizable / expandable
- Light : Server 18kb / Client 250kb
- Pure HTML and JS
- Websokets powered

Fields

- Javascript
- PWA (Progressive Web Application)
- NodeJS
- Websockets
