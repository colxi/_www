ARRAY.PROTOTYPE.SORTNUM : SORTING ALGORITHMS

Array.prototype.sortnum is an expansion of the javascript Array.prototype. It provides some sorting algorithms implementations for numerical one-dimensional arrays. It was implemented to test and analyze, performance changes in different sorting scenarios. The set of algorithms is only more performant than the native javascript sorting methods, in certain scenarios.

Available algorithms : merge sort | bubble sort | insertion sort | selection sort

The code of this experiment is hosted in Github, where it can be downloaded.
