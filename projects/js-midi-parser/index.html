MIDI-PARSER-JS : MIDI TO JSON

JSON Human readable MIDI sequences : JSMidiParser is a javascript library able to parse .mid binary data to a javascript Object data structure, ready for iteration, and interpretation. It accepts BASE64 encoded Midi data, or the resulting UINT8 arrayBuffer, obtained after reading a raw .mid binary. It also, can automatically handle files selected with a FileInput Element in the browser.
Most Standard MIDI signals are implemented, and the library provides a custom hook function to be executed when any non-standard message is found.

This library is hosted in Github, available to download, with extensive documentation.
