DEEP OBSERVER : JS OBSERVABLES

Deep-Observer is a tiny Object Observe library (100b gziped), based in ES6 Proxies, to deep watch and track changes in Objects and Arrays. The callback provided to the Constuctor is executed each time a change is deteced, recieving a complete set of data (like Object context, Property Name -or Array index-, Keypath, old value... ) relative to the detected change. Is compatible with all JS primitive types, highly configurable, and crossbrowser and NodeJS compatible.

This project, and its source code is hosted in Github, where can be downloaded and complete documentation is available.
