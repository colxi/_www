KEYPATH RESOLVE : JS LIBRARY

Keypath-resolve is a tiny and flexible library ( 1Kb gziped, 3Kb minified), for safe (any form of code evaluation is used internally) resolution and manipulation of Keypaths. Accepts keypaths formated using dot and bracket notation (or mixed). Is Extremely fast (+200,000 keypath resolutions/second) , and has wide platform support : Node , Chrome 49+, Firefox 47+, Edge, Opera , Safari 5.1+, Internet Explorer 11+

This project, and its source code is hosted in Github, where can be downloaded and complete documentation is available.
