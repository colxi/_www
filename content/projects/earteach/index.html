EARTEACH : HTML5 EAR TRAINER

EarTeach is a Musical Ear Trainer web application, implemented in HTML5. This personal project was presented as the CS50 Harvard course final project. (CS50: introduction to the intellectual enterprises of computer science and the art of programming).

The app provides a set of progresive exercises to train Intervals, Chords, Scales recognition, and let the user track its personal progress, to identify those fields that require more attention.

The source code of this project is hosted in Github available to download.
