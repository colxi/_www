JSHARMONY : MUSICAL LIBRARY

JSHarmony is a javascript library focused in generating musical objects and structures, like Chords, Intervals, and Scales. A set of secondary functions for input validation, is provided too.

Methods

- makeScale(mode,name,root) : Generate a new scale
- randomChord(root,type) : Generate a randome chord
- randomNote(name, accidentals, octave) : Generate a random note
- makeIntervalNote(grade , quality, direction, root) : Generate an internal note from the provided root
- makeInterval(grade, quality, direction, root) : Generate a interval from the provided root
- makeChord(root, type) : Generate a chord with the provided root


This project is hosted in Github, available for download, and with extensive documentation.
